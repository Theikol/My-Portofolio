## ⚙️ Installation

Clone repository:

```
git clone https://github.com/Theikol/My-Portofolio.git
```

Go to project folder:

```
cd My-Portofolio/frontend
```

Install dependencies:

```
npm install
```

---

## 🧪 Run Development Server

```
npm run dev
```

Open browser:

```
http://localhost:5173
```

---

## 📦 Build for Production

```
npm run build
```

Preview build:

```
npm run preview
```
